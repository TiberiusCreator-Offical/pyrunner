Features

• Drag & drop .py files anywhere on the page
• Renaming tabs only changes the name — extensions are locked
• Loaded files keep their original extension (.py, .tbs, etc.)
• Stats button shows code metrics + run history
• Strict mode hides errors, prints only output
• input() works inline in the output area
• Multi-tab editor with auto-save (localStorage)
• Persistent session keeps variables alive between runs
• Share Link encodes code into a compressed URL
• Auto pip install on import (numpy, pandas, etc.)
• Smart Python auto-indent after colons
